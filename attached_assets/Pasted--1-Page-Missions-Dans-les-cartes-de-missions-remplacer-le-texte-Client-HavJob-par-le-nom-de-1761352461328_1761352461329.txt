👤 4. Annuaire des freelances

Seuls les utilisateurs ayant rempli leurs informations freelances dans leur dashboard (section “Mon profil Freelance”) doivent apparaître dans l’annuaire.

Un utilisateur n’est visible dans l’annuaire que s’il a complété cette section.

🧑‍💻 5. Dashboard – Mon profil Freelance

Créer une section “Mon profil Freelance” dans le dashboard (à côté de “Favoris”).

Cette section doit permettre à l’utilisateur d’ajouter ou modifier :

Nom et prénom

Description

Liste de compétences

CV (upload PDF ou Word téléchargeable publiquement)

Ces données doivent être affichées dans l’annuaire des freelances.

⚙️ 6. Dashboard – Mes missions

Dans la modification du statut d’une mission, supprimer une des deux options suivantes :

“Ouverte”

“Toujours d’actualité”

🪪 7. Profil Freelance public

Lorsqu’un utilisateur clique sur “Voir le profil” dans l’annuaire :

Il est redirigé vers la page profil du freelance.

Cette page doit afficher :

Nom, description, compétences, CV (téléchargeable)
